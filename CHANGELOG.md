# TDS Changelog

All notable changes to the Trust Declaration Standard (TDS) will be documented in this file.

Versioning policy:
- Patch (x.y.Z): documentation or clarifications (no semantic changes)
- Minor (x.Y.z): additive changes (backward compatible)
- Major (X.y.z): breaking changes
